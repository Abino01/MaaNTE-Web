# 自動彈琴

## 簡介

讀取 MIDI 曲目並向遊戲視窗發送鍵盤演奏事件，實現自動彈琴功能。

## 功能

### MIDI 譜子

選擇要演奏的 MIDI 曲目。支援自訂路徑，也可以使用預設曲目。

### 播放速度

控制 MIDI 曲目的播放速度。 `1.0` 為原速，大於 `1.0` 加快，小於 `1.0` 減慢。

### 轉調

將 MIDI 曲目進行移調，單位為半音。例如 `-12` 降低一個八度，`12` 升高一個八度，`0` 不轉調。

## 配置詳解

### MIDI 譜子

選擇 MIDI 曲目來源。

**具體實作**：下拉選擇框 `MidiScoreFile`，預設 `自訂路徑`。選擇後提供 `MidiSettings_Custom` 子選項。

#### 自訂路徑

設定 MIDI 譜子的檔案路徑、播放速度和轉調參數。

**具體實作**：輸入框組 `MidiSettings_Custom`，包含三個輸入項目：

- **譜子路徑**（`SongPath`）：預設 `songs/example.mid`，透過 `^.+\.(mid|midi)$` 校驗。相對路徑從專案根目錄解析，也支援絕對路徑。
- **播放速度**（`Speed`）：預設 `1.0`，透過 `^(?:[1-9]\d*(?:\.\d+)?|0\.\d+)$` 校驗，需大於 0。
- **轉調**（`Transpose`）：預設 `0`，透過 `^-?\d+$` 校驗，單位為半音。

以上參數覆寫 `AutoPiano` 的 `custom_action_param` 中的 `song`、`speed`、`transpose` 欄位。
