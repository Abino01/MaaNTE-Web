# 噴泉打卡

## 簡介

自動辨識並完成噴水池打卡的每日簽到功能。

需要使用前台控制器。

## 功能

自動依序完成以下操作：

1. **識別噴泉打卡按鈕** — 透過範本匹配識別打卡交互按鈕，按 F 交互
2. **等待動畫** — 等待打卡動畫播放結束
3. **許願** — 識別許願選項並點擊，完成許願
4. **跳過劇情** — 自動跳過許願過程中的劇情對話
5. **退出** — 按 Esc 退出打卡介面

此任務無可設定選項，啟動後即完整運行全部流程。

## 配置詳解

此任務無子選項，無需額外配置。

**具體實作**：任務入口 `FountainCheckinEntrance`，無 `option` 欄位。使用模板匹配（`TemplateMatch`）識別噴泉打卡按鈕和許願選項，控制器限制為 `Win32-Front`。
