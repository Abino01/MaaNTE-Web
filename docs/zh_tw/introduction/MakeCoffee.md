# 自動做咖啡

## 簡介

自動循環進行咖啡製作小遊戲：選擇關卡、開始營業、等待銷售目標達成、領取獎勵，如此往復。

無需前台控制器。

## 功能

每輪循環執行以下步驟：

1. 點選關卡選擇區域
2. 等待"開始營業"按鈕出現並點擊
3. 重複點擊目標區域，等待銷售目標（星星圖示）達成後退出
4. 等待"領取獎勵"按鈕出現並點擊
5. 按 F 鍵繼續下一輪

## 配置詳解

### 循環次數

控制咖啡製作循環的輪數。

**具體實作**：`input` 類型輸入框 `MakeCoffeeLoopTime` ，預設值為 `10` ，透過 `pipeline_type: "int"` 校驗整數輸入。覆寫 `AutoMakeCoffee` 節點的 `custom_action_param.count` 參數實現，實際邏輯由 Python 自訂動作 `auto_make_coffee` 執行。
