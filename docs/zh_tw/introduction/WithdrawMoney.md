# 領取一咖舍收益

## 簡介

自動領取一咖舍的營業收益。

需要使用前台控制器。

## 功能

### 自動領取收益

按 F5 開啟一咖舍介面，依序點選一咖舍按鈕、提取收益、確認收益，最後關閉介面。

### 自動補貨

領完收益後，自動進行補貨作業：選擇 24 小時補貨週期、確認補貨、處理庫存已滿提示、送貨上門並確認。

## 配置詳解

### 自動補貨

"自動補貨"功能開關。關閉時只領取收益，不進行補貨操作。

**具體實現**：開關 `Restock` ，預設禁用（`No`）。啟用時將 `點擊補貨` 覆寫為 `"enabled": true` ；停用時將 `點擊補貨` 覆寫為 `"enabled": false` ，跳過補貨流程直接按 Esc 退出。
