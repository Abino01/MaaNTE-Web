# 领取一咖舍收益

## 简介

自动领取一咖舍的营业收益。

需要使用前台控制器。

## 功能

### 自动领取收益

按 F5 打开一咖舍界面，依次点击一咖舍按钮、提取收益、确认收益，最后关闭界面。

### 自动补货

在领完收益后，自动进行补货操作：选择 24 小时补货周期、确认补货、处理库存已满提示、送货上门并确认。

## 配置详解

### 自动补货

"自动补货"功能开关。关闭时只领取收益，不进行补货操作。

**具体实现**：开关 `Restock` ，默认禁用（`No`）。启用时将 `点击补货` 覆写为 `"enabled": true` ；禁用时将 `点击补货` 覆写为 `"enabled": false` ，跳过补货流程直接按 Esc 退出。
